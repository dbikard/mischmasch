# MischMasch

German-French vocabulary flashcard PWA. Built for practicing translations and noun genders (der/die/das).

## Features

- **Type mode**: type the translation from memory
- **Choice mode**: pick the correct answer from 4 options
- **Gender mode**: practice der/die/das for German nouns
- Spaced repetition weighting (struggling words appear more often)
- Paste vocabulary lists in bulk (tab, pipe, or semicolon separated)
- Works offline as an installable PWA

## Usage

Visit [dbikard.github.io/mischmasch](https://dbikard.github.io/mischmasch/) and add it to your home screen.

## Tech

Single `index.html` with React 18 from CDN, Babel standalone for JSX, localStorage for persistence. No build step.
